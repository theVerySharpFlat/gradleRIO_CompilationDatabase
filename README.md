# gradleRIO_CompilationDatabase

This will generate a `compile_commands.json` file based on a gradleRIO project.

## Usage
```bash
chmod +x ./rioCDGen.py # do this once
./rioCDGen.py
```

or

```
python3 ./rioCDGEN.py
```

## Requirements
* python3

## Platforms
* Linux
* MaxOS (hopefully)
* Windows (in theory)
